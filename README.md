# my-portfolio

丸山唯成（Yuisei Maruyama）のポートフォリオサイト。

## Tech Stack

- **Framework**: Next.js 16 (App Router, Turbopack)
- **Language**: TypeScript
- **CSS**: TailwindCSS 4
- **i18n**: next-intl（日本語 / 英語）
- **Animation**: Framer Motion
- **Package Manager**: pnpm

## Development

```bash
make dev        # Start dev server
make build      # Production build
make lint       # ESLint
make typecheck  # TypeScript check
```

## Deployment

- **Platform**: [Vercel](https://vercel.com/yuiseidev-4722s-projects/yuisei-maruyama)
- **URL**: https://yuisei-maruyama.vercel.app
